# Planning Artifacts

## CONTEXT.md

Project context, requirements, and stakeholder information.

**Sections:**
- Project Overview
- Goals and Objectives
- Stakeholders
- Constraints
- Assumptions
- Reference Documents
- Source Code Locations (if applicable)

## PROGRESS.md

Current status, completed work, and next steps.

**Sections:**
- Current Phase
- Completed Items
- In Progress Items
- Next Steps
- Open Questions
- Blockers

## sessions/YYYY-MM-DD.md

Session notes documenting work performed each day.

**Sections:**
- Date
- Focus Area
- Work Completed
- Decisions Made
- Next Session

## research/[topic].md

Research findings on specific topics.

**Sections:**
- Date, Status, Purpose
- Research Question
- Executive Summary
- Scope (In/Out)
- Findings
- Trade-off Analysis (table)
- Risks (table)
- Recommendations
- Planning Implications
- Open Questions
- References

## roadmap/ or deliverables/

Phase-specific planning outputs (roadmaps, charters, WBS, etc.)
